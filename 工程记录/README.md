## 当前分类入口的最新读法

当前二阶段公开叙事已经不应只看 `Tensor-CSPNet`。

- `Tensor-CSPNet`
  - 当前主要承担 EEG/SPD 宿主验证
  - 其价值仍然很高，但已经不是唯一公开结果入口
- `ResNet1D`
  - 当前已经形成更完整的多数据集结果沉淀
  - 是通用 MTS 宿主的当前主结果入口

因此如果你现在要接住分类工程，默认顺序应是：

1. `分类工程现状`
2. `分类调试记录`
3. 再按需进入 `Tensor-CSPNet` 任务单和阶段一证据
