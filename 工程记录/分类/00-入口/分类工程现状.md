## 1. 当前分类工程的总结构

当前分类工程最准确的理解方式不再是“一条主线”，而是四层并存：

1. **二阶段外部宿主验证层**
2. **一阶段内部几何证据层**
3. **并行分类支线**
4. **历史背景层**

当前最重要的变化是：

- 当前真实在推进的默认实验，已经从一阶段切到二阶段
- 一阶段没有失效，但角色从“当前方法载体”切成了“结构证据来源”

## 2. 当前活跃层：二阶段外部宿主验证

### 2.1 当前默认路线

当前活跃路线固定为：

`Tensor-CSPNet backbone -> Temporal_Block 后 latent -> local closed-form residual head -> residual fusion`

当前宿主：

- `Tensor-CSPNet`

当前协议：

- `BCIC-IV-2a holdout`

当前实验矩阵：

- `E0`
  - 原始 `Tensor-CSPNet`
- `E1`
  - `Tensor-CSPNet + residual linear adapter`
- `E2`
  - `Tensor-CSPNet + local closed-form residual head`

### 2.2 当前目标

当前二阶段不是直接回答“是否超越所有旧主线”，而是先回答：

1. 宿主 baseline 是否可靠复现
2. `E2` 是否优于普通 residual 对照 `E1`
3. `E2` 是否优于宿主 `E0`

因此当前二阶段的默认比较契约是：

- `Tensor-CSPNet` 原始宿主
- `residual linear` 最小结构对照
- `local closed-form residual` 方法主臂

### 2.3 当前已经完成到哪一步

当前已完成：

- `Tensor-CSPNet` 外部宿主复现：
  - `BCIC-IV-2a holdout = 0.7238`
- `E0 / E1 / E2`
  - `subject 1 / 1 epoch / seed 1` 最小 smoke

当前状态：

- 最小 smoke 已经说明：
  - wrapper 已通
  - residual 融合已通
  - 局部闭式残差头的 forward / backward 已通
- 当前正在做：
  - `BCIC holdout` 单 seed 全量比较

## 3. 当前证据层：一阶段内部几何主线

### 3.1 当前身份

一阶段当前不再承担“当前默认对外方法形态”的职责，而主要承担：

- 局部几何与闭式算子的内部证据层
- 二阶段设计的结构来源
- 失败分支与有效分支的归因来源

### 3.2 一阶段当前最重要的结论

#### `P0a.1`

当前已经站住的不是完整快慢闭环，而是：

**冻结慢层下的快层 operator family**

核心有效部件：

- `A2r`
- `C3LR`
- `r > 1`
- `optional B3`

#### `R0 / P0b-lite`

当前结论：

- 有局部正信号
- 但不是跨数据集稳定赢家

因此只保留为：

- 机会分支
- 结构拆解分支

#### `P1a`

当前最重要的结构信号是：

- 继续修全局固定算子不够
- 更值得推进的是：
  - `query-conditioned local operator`
