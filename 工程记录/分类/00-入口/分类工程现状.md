## 1. 当前分类工程的总结构

当前分类工程最准确的理解方式不再是“一条主线”，而是四层并存：

1. **二阶段外部宿主验证层**
2. **一阶段内部几何证据层**
3. **并行分类支线**
4. **历史背景层**

当前最重要的变化是：

- 当前真实在推进的默认实验，已经从一阶段切到二阶段
- 一阶段没有失效，但角色从“当前方法载体”切成了“结构证据来源”

## 2. 当前活跃层：二阶段外部宿主验证

### 2.1 当前默认路线

当前二阶段已经从“单一 Tensor-CSPNet 宿主”扩展成双宿主结构：

- `Tensor-CSPNet`
  - 当前主要承担 EEG/SPD 外部宿主验证
  - 当前协议仍以 `BCIC-IV-2a holdout` 为主
- `ResNet1D`
  - 当前主要承担通用 MTS 宿主验证
  - 当前协议是 fixed-split 多数据集框架对比

二阶段当前统一的框架矩阵是：

- `E0`
  - backbone + global linear head
- `E1`
  - backbone + residual linear head
- `E2`
  - backbone + DLCR

### 2.2 当前目标

当前二阶段的公开目标已经分成两块：

1. `Tensor-CSPNet`
   - 验证端到端局部闭式残差头能否在 EEG/SPD 宿主上稳定落地
2. `ResNet1D`
   - 验证 `DLCR` 是否能在通用 MTS 宿主上形成真实收益

因此当前二阶段默认比较契约统一为：

- `E0`
  - backbone + global linear head
- `E1`
  - backbone + residual linear head
- `E2`
  - backbone + DLCR

### 2.3 当前已经完成到哪一步

当前已完成的阶段二结果，可以分成两组。

#### `Tensor-CSPNet`

- 外部宿主复现：
  - `BCIC-IV-2a holdout = 0.7238`
- `E0 / E1 / E2` 已完成单 seed 全量比较
- 当前阶段性读法：
  - `SPD fp32` 已经是可用候选数值策略
  - `E2` 目前还没有稳定优于 `E0 / E1`

#### `ResNet1D`

- `E0 / E1 / E2` 第一版框架对比已覆盖 9 个数据集：
  - `NATOPS`
  - `FingerMovements`
  - `SelfRegulationSCP1`
  - `BasicMotions`
  - `HandMovementDirection`
  - `UWaveGestureLibrary`
  - `Epilepsy`
  - `AtrialFibrillation`
  - `PenDigits`
- 第一版对比的关键读法：
  - `E2` 在 `FingerMovements` 上优于 `E0 / E1`
  - `E1` 在 `SelfRegulationSCP1 / UWaveGestureLibrary / Epilepsy / PenDigits` 上很强
  - `E2` 不是统一赢家，但已在通用宿主上出现真实正信号
- `E2 + pinv` 已完成对 `E0` 的 9 数据集全量比较
- 模板矩阵已完成两条高信息量对照：
  - `1模板 + pooled`
  - `4模板 + committee_mean`

### 2.4 当前阶段二的公开读法

到当前更新时间为止，阶段二不应再被理解成：

- 只有 `Tensor-CSPNet` 一条宿主线
- `ResNet1D` 只是代码预备态
- 公开结果沉淀仍主要停留在 `Tensor-CSPNet`

更准确的说法是：

- `Tensor-CSPNet`
  - 仍是重要宿主
  - 但更偏“外部 EEG/SPD 宿主验证”
- `ResNet1D`
  - 已经形成当前更完整的通用 MTS 结果沉淀
  - 当前公开叙事应该把它提升到与 `Tensor-CSPNet` 同级

## 3. 当前证据层：一阶段内部几何主线

### 3.1 当前身份

一阶段当前不再承担“当前默认对外方法形态”的职责，而主要承担：

- 局部几何与闭式算子的内部证据层
- 二阶段设计的结构来源
- 失败分支与有效分支的归因来源

### 3.2 一阶段当前最重要的结论

#### `P0a.1`

当前已经站住的不是完整快慢闭环，而是：

**冻结慢层下的快层 operator family**

核心有效部件：

- `A2r`
- `C3LR`
- `r > 1`
- `optional B3`

#### `R0 / P0b-lite`

当前结论：

- 有局部正信号
- 但不是跨数据集稳定赢家

因此只保留为：

- 机会分支
- 结构拆解分支

#### `P1a`

当前最重要的结构信号是：

- 继续修全局固定算子不够
- 更值得推进的是：
  - `query-conditioned local operator`
