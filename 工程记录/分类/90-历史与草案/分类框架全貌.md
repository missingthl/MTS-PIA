## 1. 这份文档现在还回答什么

它现在主要回答三件事：

1. 一阶段早期分类工程从 `trial/raw` 到 `z-space` 是怎么组织的
2. 为什么旧 `Route B / unified shell` 后来被降级
3. 为什么当前方法学核心最终收回到了闭式算子与局部几何上

## 2. 一阶段历史链条的最短概括

旧的一阶段历史链条可以压成：

`dataset trials -> protocol split -> SPD/log-Euclidean z-space -> legacy Route B / unified shell / PIA Core recovery -> bridge -> raw MiniROCKET`

这条链对现在仍有价值的地方不是“照搬实现”，而是它留下了三类结构资产：

- `z-space / SPD / log-Euclidean` 表示经验
- `TELM2 / closed-form operator` 的可解释性资产
- `bridge / terminal evaluator` 的工程分层经验
