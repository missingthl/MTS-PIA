# Dynamic Manifold Evolution Plot

This figure tracks the same original train trajectories across four dynamic-stage representations:

- baseline
- T2a default
- T2b-0 saliency-aware
- T3 rebasis

The plot uses a shared embedding across stages, plus a centroid-evolution panel.
This is closer to the current dynamic branch than the older static manifold scatter and can be read as a manifold-evolution view.
