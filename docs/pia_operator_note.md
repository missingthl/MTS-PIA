# PIA Operator Note

更新时间：2026-03-27

当前必须把 `PIA` 本体明确区分为两层：

1. 模板学习器
   - `TELM2Transformer`
   - 只用训练集
   - 通过闭式/岭回归式回写学习 `W, b`
2. 增强算子
   - 不是 `z + gamma * d`
   - 而是以训练均值为中心，对模板轴投影分量做显式仿射拉伸，并保留正交残差

当前最小算子语义固定为：

`x' = mu + sum_k (1 + gamma_k) * proj_k(x - mu) + residual(x - mu)`

其中：

- `proj_k` 是样本在第 `k` 条模板轴上的投影分量
- `residual` 是去掉所选模板轴投影后的正交残差

第一阶段默认只做单主模板拉伸：

- `gamma = [0.1, 0, 0, ...]`

多轴并发仿射是后续扩展，不作为当前最小恢复目标。
