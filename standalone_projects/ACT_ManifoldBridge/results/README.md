# Release Results

This directory intentionally contains only compact release-facing summaries.

- `release_summary/main_comparison.csv`: the primary comparison table for
  baseline CE, original MBA, RC-4, and zPIA-top1.
- `release_summary/internal_ablation_reference.csv`: compact internal reference
  table used to document why zPIA-top1 is the current default.

Large raw experiment directories, audits, router traces, wavelet experiments,
progressive pilots, and historical paper-asset sweeps are not kept in the
release tree.
